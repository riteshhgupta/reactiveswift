# ProtoKit

ProtoKit is a protocol oriented UIKit!

## Installation

### Cocoapods
To integrate ProtoKit into your Xcode project using CocoaPods, specify it in your Podfile:

```
pod 'ProtoKit', :git => 'https://github.com/riteshhgupta/ProtoKit.git'
```
